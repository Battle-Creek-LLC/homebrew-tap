# Battle Creek LLC — Homebrew Tap

Homebrew formulae for Battle Creek LLC's open-source CLIs.

## Usage

```sh
brew tap Battle-Creek-LLC/tap
brew install <formula>
```

Or in one step:

```sh
brew install Battle-Creek-LLC/tap/<formula>
```

## Formulae

| Formula | Description | Upstream |
|---|---|---|
| `moot` | Send a Recall.ai bot to a meeting, capture the transcript, generate notes | [Battle-Creek-LLC/moot](https://github.com/Battle-Creek-LLC/moot) |
